# ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ MVP:

Указанные версии ПО - актуальные на текущий момент (Октябрь 2025). Разрешается использовать более новые стабильные версии ПО.

## Окружение и запуск
- **ОС и среда**: Ubuntu 24.04 LTS production и Ubuntu 24.04 LTS в WSL на Windows для разработки и тестирования.
- **Запуск**: ручной (без планировщиков, демонов и автозапуска в рамках MVP).
- **Дистрибуция**: PROD: Docker Compose, DEV: локальный запуск.

## Стек и версии
- **n8n**: 
- **Python**: 3.13.7 и новее (обязательно использование `venv`).
- **Playwright for Python**: 1.55 и новее (браузер Chromium, headed-режим и CDP).
- **Библиотеки**: `aiofiles 24.1.0`, `Pillow 11.3.0`.
- **Запреты**: не использовать API платежных систем; только полнофункциональный браузер.
- **Имитация человека**: случайные задержки, мягкие клики, естественная прокрутка, паузы на «осмотр», задержки печати текста.
